## Build Process 

To build the application : ``mvn clean package``

Package **HelloWorldApp-1.0-SNAPSHOT.jar** will be created at the location **target/**

To run the application : ``java -jar target/HelloWorldApp-1.0-SNAPSHOT.jar``
